# Address API SPEC 

## Create Address
Endpoint: POST /api/contacts/{contactID}/addresses

Request Headers: 
- Authentication: token

Request Body:
```json
{
    "street":"street",
    "city":"city",
    "province":"country",
    "country":"country",
    "postal_code":"1512"
}

```
Response Body: 
```json
{
    "data":{
        "id":1,
        "street":"street",
        "city":"city",
        "province":"country",
        "country":"country",
        "postal_code":"1512"
    }
}
```
